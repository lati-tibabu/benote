## Workspace

The Workspace section allows users to manage their projects, tasks, and study plans. It includes the following components:

### Add New Workspace

- **File**: `add_new.jsx`
- **Description**: Enables users to create a new workspace.

### Opened Workspace

- **File**: `workspace_opened.jsx`
- **Description**: Displays the details of an opened workspace, including tasks, notes, and roadmaps.

#### Sub-components:

1. **Roadmaps**

   - **File**: `roadmaps.jsx`
   - **Description**: Displays a list of roadmaps associated with the workspace.

2. **AI-Generated Roadmap**

   - **File**: `ai-generated-roadmap.jsx`
   - **Description**: Provides AI-generated suggestions for roadmaps.

3. **Tasks**

   - **File**: `tasks.jsx`
   - **Description**: Manages tasks within the workspace.

4. **Notes**

   - **File**: `notes.jsx`
   - **Description**: Allows users to create and manage notes.

5. **Settings**
   - **File**: `settings.jsx`
   - **Description**: Configures workspace-specific settings.
